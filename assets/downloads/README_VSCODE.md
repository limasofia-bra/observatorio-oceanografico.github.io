# Perfil VS Code - O2 Student Starter + Copilot + Excel Viewer

## Arquivo

`o2-student-starter-copilot-excel.code-profile`

## Finalidade

Este perfil foi preparado para estudantes do O2 para acelerar a configuracao inicial do Visual Studio Code com extensoes e ajustes basicos para atividades academicas e tecnicas.

O perfil inclui suporte a Git/GitHub, Markdown, Python, Jupyter, YAML, Quarto, CSV, visualizacao de Excel e GitHub Copilot.

## Como importar no VS Code

1. Baixe o arquivo `.code-profile`.
2. Abra o VS Code.
3. Clique na engrenagem no canto inferior esquerdo.
4. Entre em `Profiles`.
5. Selecione `Import Profile`.
6. Escolha o arquivo baixado.
7. Ative o perfil importado.
8. Faca login no GitHub dentro do VS Code para usar o Copilot.

## Regras de seguranca

- O Copilot deve ser usado com a conta GitHub individual de cada aluno.
- O perfil nao fornece acesso as maquinas do laboratorio.
- O perfil nao inclui credenciais.
- O perfil nao inclui chaves SSH.
- O perfil nao deve conter dados sensiveis.
