# Downloads

## Finalidade da pasta

A pasta `assets/downloads` armazena arquivos estaticos publicos para download usados ou referenciados pelo site institucional do Observatorio Oceanografico.

No contexto do GitHub Pages/Jekyll, esta pasta nao e uma colecao Jekyll e nao deve conter paginas Markdown destinadas a navegacao do site.

## Arquivos disponiveis

- `o2-student-starter-copilot-excel.code-profile`

Para instrucoes detalhadas do perfil VS Code, consulte `README_VSCODE.md` nesta mesma pasta.

## Cuidados

- nao adicionar senhas;
- nao adicionar tokens;
- nao adicionar chaves SSH;
- nao adicionar arquivos privados;
- nao adicionar dados administrativos sensiveis;
- nao adicionar arquivos grandes sem necessidade;
- preferir nomes de arquivos curtos, sem acentos e com hifens.

## Convencao de nomes

Use nomes em minusculas, sem espacos, sem acentos e com hifens.

Exemplos:
- `o2-student-starter-copilot-excel.code-profile`
- `modelo-cadastro-amostras.pdf`
- `guia-github-alunos.pdf`

## Observacao final

Paginas explicativas devem ficar nas colecoes ou paginas apropriadas do site.

Esta pasta deve guardar apenas arquivos baixaveis, publicos, reutilizaveis e seguros para exposicao no repositorio.

## Como contribuir com o site

As instrucoes para criar ou atualizar paginas pessoais, trabalhar com branches e enviar pull requests estao no arquivo [CONTRIBUTING.md](../../CONTRIBUTING.md) da raiz do repositorio.
